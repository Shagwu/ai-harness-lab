# AI Harness Lab

This is a "Superhero Suit" for AI agents. It helps them work safely and stay organized.

## Components
- **Playroom (Docker):** Isolated workspace.
- **Toolbox (src/tools.py):** List of robots the AI can call.
- **Secretary (Context):** Keeps track of the plan.
- **Boss (Orchestrator):** Manages the To-Do list.
